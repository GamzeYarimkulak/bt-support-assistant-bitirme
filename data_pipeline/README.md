# Data Pipeline

Bu klasör, BT destek asistanı projesinde veri ön işleme süreçlerini içermektedir.

## Amaç
Destek kayıtlarının sistemde kullanılmadan önce temizlenmesi, düzenlenmesi ve anonimleştirilmesi hedeflenmektedir.

## Bu aşamada yapılanlar
- Örnek veri seti oluşturuldu
- Temel anonimleştirme fonksiyonları eklendi
- Veri işleme süreci planlandı

## Anonimleştirme
Veri güvenliği nedeniyle e-posta, telefon, IP ve isim gibi hassas bilgiler maskelenmektedir.

## Sonraki adımlar
- Metin temizleme
- Chunking
- İndeksleme
- Retrieval için veri hazırlama
