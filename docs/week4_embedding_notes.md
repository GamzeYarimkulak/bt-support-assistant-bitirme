# Hafta 4 - Embedding Tabanlı Retrieval Notları

Bu hafta proje kapsamında embedding tabanlı semantik arama altyapısı üzerine çalışılmıştır.

## Yapılanlar
- Kelime temelli BM25 aramaya ek olarak semantik arama ihtiyacı netleştirildi
- Embedding tabanlı retrieval modülü projeye eklendi
- Dokümanların vektör temsilleri üzerinden benzerlik hesaplama yaklaşımı hazırlandı
- Retrieval değerlendirmesi için temel metrikler düzenlendi

## Amaç
Kullanıcı sorgularının yalnızca kelime eşleşmesine göre değil, anlam benzerliğine göre de değerlendirilebilmesini sağlamak.

## Sonraki Adım
